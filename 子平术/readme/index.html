· 目前主要参考自《中国古代星占学》、《中国命理学史论》、《中国古代历法》、维基、百度

· 占卜从最初的观测星象发展到后来的仪器模型模拟天象，进而脱离仪器开始“公式化”：太乙、遁甲、六壬。但所用规则确是星占时传下的，而星象早变了。像 子平术这种没那么“星占”但依赖时间维度的体系，随着历法的计算方式修改，在日用上准确度提升，但在占卜上是否该用古历法（以星象确立）排盘？

· “干表气，支表时，及其五行配属"的含义是源自中原的季节和气候变化，这是否代表着阴阳干支五行这套体系只适用于中原地区🤔️？子平术的盘从生辰排出， 缺乏地理空间维度，面对各地在同节气中的不同气候表现，得令生旺的含义有普适性吗？
